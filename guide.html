Windows Install Guide

  1. Download and Install Java 8 (64-bit) Here

  2. Download and Install MultiMC Here

  3. Download our modpack from Here

  4. Run MultiMC and go through the initial setup, making sure to choose Java 8 64bit as your java version.

  5. Right click in the open are and click 'Create Instance'

  6. Click import from Zip, and use browse to navigate to the zip file.

  7. MultiMC may freeze, but it is still importing the modpack, be patient :3

  8. Launch the Instance, Go to Multiplayer and Join!

    9. **If your game fails to launch, sometimes you need to go back to the instance settings and choose 64bit java manually, I recommend allocating 8192MB of System Memory**

Debian/Ubuntu/PopOS! Install Guide

  1. Install Java 8 by running 'sudo apt-get install openjdk-8-jdk' in terminal

  2. Install PolyMC by running the following commands in terminal.

    • curl -OL https://github.com/PolyMC/PolyMC/releases/download/1.4.3/PolyMC-1.4.3.flatpak

    • flatpak install --system PolyMC-1.4.3.flatpak

    • rm PolyMC-1.4.3.flatpak

  3. Download our modpack from Here

  4. Run PolyMC and go through the initial setup, making sure to choose Java 8 64bit as your java version.

  5. Right click in the open are and click 'Create Instance'

  6. Click import from Zip, and use browse to navigate to the zip file.

  7. PolyMC may freeze, but it is still importing the modpack, be patient :3

  8. Launch the Instance, Go to Multiplayer and Join!

    9. **If your game fails to launch, sometimes you need to go back to the instance settings and choose 64bit java manually, I recommend allocating 8192MB of System Memory**
